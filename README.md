## Overview

- Static portfolio site built with HTML
- Includes links to project pages and basic portfolio sections
- Displays preview images for two projects

## Projects

- Movie Ranking Project
- Birthday Invite Project

## Structure

- `index.html` – main portfolio page
- `public/movie-ranking.html` – movie ranking project page
- `public/birthday-invite.html` – birthday invite project page
- `public/about.html` – about page
- `public/contact.html` – contact page
- `assets/images/` – project preview images

## Usage

1. Open `index.html` in a web browser
2. Click project links to navigate
